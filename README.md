# 달보드레 by 개발자만 넷
## ☝ Description
달보드레는 사용자의 고민을 해결하기 위해 매일 랜덤한 문장을 사용자에게 주는 앱입니다. <br />
Dalvodre is an app that gives users random sentenses every day to solve the their problems.
